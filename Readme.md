
**Prueba-de-desempe-o-Modulo-HTML-CSS**
-----------------------------------------------------------------------

This website is made with the purpose of imitating the page created by TL Moises

---------------------------------------------------------------------

DEMO Github link (Where all the code is stored):
 https://github.com/OneBolivar/Prueba-de-desempe-o-Modulo-HTML-CSS.git

-----------------------------------------------------------------------

TECHNOLOGIES USED

    HTML5: Semantic structure of content.
    CSS3: Advanced styles and layouts.
    Sass: Preprocessor for a modular and maintainable style architecture.
    JavaScript: Client-side logic and interactivity.

-----------------------------------------------------------------------

FEATURES

    Mobile First design and fully responsive.
    Modular styles organized with Sass.
    Dynamic DOM manipulation using JavaScript.

-----------------------------------------------------------------------

Installation and Local Use If you want to clone this project and run it on your local machine, follow these steps:

    Clone the repository: git clone https://github.com/OneBolivar/Prueba-de-desempe-o-Modulo-HTML-CSS.git

    Enter the folder: cd project-name

    Compile Sass (if necessary): If you don't use the Live Sass Compiler extension for VS Code, run: sass --watch scss/main.scss:css/style.css

    Open the project: Simply open the index.html file in your favorite browser.
